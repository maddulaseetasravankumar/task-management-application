# Task Manager Application

A modern and responsive task management application developed using React.js and Material UI. This application helps users organize daily tasks efficiently with a clean and user-friendly interface.

## Features

- Add new tasks
- Delete tasks
- Mark tasks as completed
- Responsive and modern UI
- Reusable React components
- Smooth user experience

## Technologies Used

- React.js
- JavaScript (ES6)
- Material UI
- CSS3
